Detailed Description

Compute \(m^{mth}\) Fibonacci number using the formulae:

Author

\begin{eqnarray*} F_{2n-1} &=& F_n^2 + F_{n-1}^2 \\ F_{2n} &=& F_n\left(2F_{n-1} + F_n\right) \end{eqnarray*}
